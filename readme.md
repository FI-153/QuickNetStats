# QuickNetStats
### Essential network statistics directly in the Menu Bar

![153](https://github.com/FI-153/QuickNetStats/blob/d02c41969868b0fdc06dc63916e5cbc0f0d13c8f/Assets/main.png)

## Installation
You can install this easily using Homebrew as a cask.
### 1. Add the tap
```bash
brew tap FI-153/tap
```
### 2. Install the cask
```bash
brew install quicknetstats
```
